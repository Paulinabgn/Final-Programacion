Creacion de un Sistema de Reclamos Universitarios, por medio de una interfaz web creada en Flask. Este sistema deja que acceda un usuario existente o crear uno nuevo, pueden acceder tanto alumnos, como profesores y mas. Dentro de menu de opciones del usuario se puede crear un reclamo, ver estado del mismo, ver reclamos de un mismo usuario, adherirse a reclamos existentes, cambiar el reclamo de departamento. Siendo profesor u otros, puede acceder a ver los reclamos de un determinado departamento, cambiar el estado del reclamo, etc.
