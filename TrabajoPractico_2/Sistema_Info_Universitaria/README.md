### Descripción Proyecto 1

Creacion de una Trivia de películas, por medio de una interfaz web creada en Flask. Este juego consiste en la elección de una cantidad de intentos, luego aleatoriamente se muestra una frase y el usuario debe elegir la pelicula, a la que pertenece dicha frase, dentro de una serie de opciones. Juagara tantos intentos haya elegido y una vez terminada la partida podra ver su puntaje, acceder a graficas y listas de peliculas.
